Role Summary:
A budding Web Developer to join our dynamic team in creating robust and user-friendly web applications. This role is perfect for someone with a basic understanding of frontend and backend technologies aiming to grow into more complex roles within the company. The ideal candidate will also possess strong problem-solving skills, willingness to learn continuously, and an interest in team collaboration.
Key Responsibilities:
- Develop responsive web applications with HTML5/CSS3 using Python's Django framework for backend operations while ensuring high performance standards by optimizing code efficiency and maintainability.
- Manage database design and SQL queries to facilitate fast data retrieval, which enhances user experience within the application environment.
- Implement frontend functionalities utilizing HTML5/CSS3 along with JavaScript libraries for creating engaging and interactive web interfaces that provide an outstanding visual appeal on various devices.
- Construct API endpoints using Flask framework to connect diverse systems in a seamless manner, fostering better communication within the organization's technological infrastructure.
Required Qualifications:
A bachelor’s degree or higher from an accredited institution with at least 0-1 years of relevant web development experience would be favorable for this role; however, we value practical knowledge and potential as much as formal education in our hiring process. Experience working within a team setting is highly desired but not mandatory due to the hands-on nature of onboard training provided by experienced developers during your tenure with us.
Role Constraints:
The candidate should have demonstrable expertise or willingness to learn Python, Django, SQL (with an emphasis on PostgreSQL), HTML5/CSS3, JavaScript (ES6+), and Flask; Git for version control alongside Docker skills would be advantageous but not a strict requirement due to the collaborative nature of our work environment.
